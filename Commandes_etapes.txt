git add *
Ca veut dire: Tout les fichiers modifiés et ajoutés, on les index.
Maintenant, il faut les empaqueter:
git commit -m 'initial commit'
En fait 'initial commit' , c'est le commentaire du commit, le premier s'appelle ainsi.
Pour forcer l'indexation d'un fichier spécifique pas passé en vert:
git add .gitignore
git commit -m 'Ajout du GitIgnore'

Pour la première fois qu'on push, on doit préciser:
git push --set-upstream origin master
